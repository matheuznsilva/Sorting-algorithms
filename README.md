## Instalação

Todas as instalações abaixo são referentes ao Ubuntu

### Install GCC

O GCC geralmente já está instalado no Ubuntu, mas você pode verificar a versão digitando:

     $ gcc --version

Se não estiver instalado, você pode instalar usando o apt:

     $ sudo apt update
     $ sudo apt install build-essential

## Uso

Na main da sua aplicação adicione a biblioteca do algoritmo de ordenação utilizado 
    
    #include "Diretorio/algoritmo_de_ordenação.h" 

Obs.: realize as modificações necessárias no makefile para a compilação da biblioteca acima.
